# First Release Notes

Use this as the starting point for the first App Store release notes.

## What’s New

Welcome to Ultimate Golf Community.

This first release helps you:

- Post and discover tee times
- Join local golf groups
- Build connections with other golfers
- Keep your golf profile, home course, and activity in one place

We are just getting started, and more features around messaging, groups, scores, and community activity will keep improving from here.
