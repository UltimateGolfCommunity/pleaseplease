# Screenshot Plan

This checklist is for the first App Store Connect screenshot set for the iOS app in [`mobile-app`](/Users/lukerestall/Documents/GitHub/pleaseplease/mobile-app).

## Recommended screenshot set

1. Home
- Weather
- Your next tee time
- Tee times / network feed view

2. Groups
- Groups discovery
- My groups
- Create group button visible

3. Group Detail
- Group cover and logo
- Members tab with add-member flow
- Board tab with conversation

4. Connections
- Search golfers
- Accepted connections
- Pending requests

5. Profile
- Profile header
- Activity
- Edit/share controls

6. Tee Time Creation
- Floating compose button
- Post tee time form

## Screenshot notes

- Use a clean seeded account with realistic golf data
- Remove broken placeholder text before capture
- Avoid alert popups or error banners in screenshots
- Use the same visual style and data tone across the whole set
- Prefer consistent branding and one visual theme

## Device targets

Capture at minimum:
- 6.7-inch iPhone
- 6.5-inch iPhone if needed for compatibility

## Pre-capture checklist

- Sign in works
- Weather loads
- Group images persist
- Home course saves
- Connections load
- Tee times display
- Notifications and messages open
